Iris Analyzer is an AI-powered application that performs deep analysis of iris images to uncover biometric patterns associated with health indicators and identity. Built with Streamlit, it offers an intuitive interface for uploading and analyzing iris photographs in real time.

Try it live at: [iris-analyzer.streamlit.app](https://iris-analyzer.streamlit.app)
